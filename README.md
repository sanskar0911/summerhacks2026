## 🛠️ Technology Stack

### Frontend: The High-Density UI
- **Framework**: [React 19](https://react.dev) + [TanStack Start](https://tanstack.com/start)
- **Routing**: [TanStack Router](https://tanstack.com/router) (SSR Optimized)
- **Styling**: Vanilla CSS + [TailwindCSS 4](https://tailwindcss.com) (Modern utility engine)
- **Animations**: [Framer Motion](https://framer.com/motion) (Fluid glassmorphic interactions)
- **State**: [TanStack Query](https://tanstack.com/query) (Neural data synchronization)

### Backend: The Orchestration Layer
- **Runtime**: [Node.js](https://nodejs.org) + [Express 5](https://expressjs.com)
- **AI Model**: [Google Gemini 1.5 Flash](https://deepmind.google/technologies/gemini/) (Low-latency strategic intelligence)
- **Database**: [Sequelize](https://sequelize.org) + [SQLite](https://sqlite.org) (Auto-seeding "Elite" records)
- **Communication**: [Nodemailer](https://nodemailer.com) (Integrated Mail Engine)

---

## 🚀 Professional Deployment

AgentOS is architected for seamless cloud synchronization:

- **Frontend (Vercel)**: Deployed as a **Node.js SSR** application. All routes are handled server-side to ensure zero 404s and optimal SEO performance.
- **Backend (Render)**: Deployed as a secure Web Service. Native binaries (`sqlite3`) are built from source on the Render environment to ensure 100% hardware compatibility.

## 💻 Local Calibration

To initialize the AgentOS environment locally:

### 1. Synchronize the Repository
```bash
git clone https://github.com/sanskar0911/summerhacks2026.git
cd summerhacks2026
```

### 2. Configure Intelligence (Backend)
```bash
cd backend
npm install
# Create a .env file with the following:
GEMINI_API_KEY=your_key_here
JWT_SECRET=nexus_protocol_key
EMAIL=your_email@gmail.com
APP_PASSWORD=your_app_password
```

### 3. Initialize the Command Centre (Frontend)
```bash
cd ../frontend
npm install
npm run dev
```
